1. pip install virtualenv
2. create a virtual environmental
	virtualenv env
3. activate
	cd env/scripts
	activate
4. to see already installed libraries
	pip list
5. install other libraries
	pip install pandas numpy matplotlib
6. install jupite notebook
	pip install jupyter
7. open jupyter notebook (in project folder within (env))
	jupyter notebook


to create requirements file (in project directory)
	pip freeze > file_name.txt
